# 天気情報をもとに服装を提案アプリ

このアプリは、天気予報の情報をもとに
その日の服装の目安を提示するツールです。

気温や天候を毎回確認・判断する手間を減らし、
日常の意思決定をサポートします。

想定利用シーン：
・外出前の服装判断
・業務開始前の身支度確認
・天候に左右されやすい現場作業の準備

使用技術：
・Python
・外部天気API
・Streamlit
